# TwizzyCraft 3
Twizzy Craft is a modpack filled with adventures, new creatures to discover, machines for all sorts of things, magic to manipulate the world around you, and so much more!

## Included in the Pack:
### Tech mods
- Create
- Mekanism
- Industrial Foregoing
- **Oritech**
- Powah
- AE2

### Magic Mods
- Ars Noveau
- **Spectrum**
- Occultism
- **Malum and Gaze**
- **Iron's Spells n' Spellbooks**
### Misc:
- Silent Gear
- Apotheosis
- Mystical Agriculture
- Sophisticated Storage & Backpacks

AND MUCH MORE!
